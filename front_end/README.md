# Trekking App - Flutter Frontend

## Description
Frontend application for the Nepal Trekking App built with Flutter.

## Features
- Browse trekking destinations
- View interactive maps with trek routes
- Real-time weather information
- Destination-based group chat
- Booking management
- User authentication

## Setup
1. Install Flutter SDK
2. Run `flutter pub get`
3. Run `run-app.ps1` so the app uses the current LAN IP automatically
4. If you run Flutter manually, pass `--dart-define API_BASE_URL=http://<your-ip>:8000/api`
