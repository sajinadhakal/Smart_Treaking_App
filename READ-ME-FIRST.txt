⚡ WHAT THIS PROJECT DOES:

   ✅ Full-stack Trekking App (Flutter + Django)
   ✅ Browse Nepal trekking destinations
   ✅ Interactive route maps (OpenStreetMap)
   ✅ Real-time weather & risk alerts  
   ✅ Group chat per destination
   ✅ Booking system
   ✅ Reviews & ratings
   ✅ Admin management panel

   4. Show features to your professor!

💻 TECH STACK:

   Frontend:  Flutter (Dart)
   Backend:   Django REST Framework (Python)
   Database:  SQLite
   Maps:      OpenStreetMap
   Chat:      WebSockets (Django Channels)
   Security:  PBKDF2-SHA256 password hashing

   Not sure? Run: CHECK-REQUIREMENTS.bat

🚦 FIRST TIME SETUP:

   1. Double-click START-APP.bat
   2. It will automatically:
      ✓ Check prerequisites
      ✓ Install packages
      ✓ Setup database
      ✓ Ask for admin password (remember it!)
      ✓ Load sample data
      ✓ Start servers
   3. Takes 5-10 minutes (only once!)
   4. After that, starts in 30 seconds!

═══════════════════════════════════════════════════════════════════════════════

🎮 AFTER STARTING:

   Two windows will open:
   
   1. Backend Server   → Keep running (green text)
   2. Flutter App      → Your app runs here
   
   Access Points:
   - App: On your phone
   - Admin: http://YOUR-IP:8000/admin
   - API: http://YOUR-IP:8000/api/

═══════════════════════════════════════════════════════════════════════════════

⚠️ COMMON ISSUES:

   "No devices detected"
   → Connect phone via USB, enable USB debugging
   
   "Python not found"  
   → Install Python, check "Add to PATH" during install
   
   "Backend won't start"
   → Check Windows Firewall, allow port 8000
   
   More solutions: TROUBLESHOOTING.md
