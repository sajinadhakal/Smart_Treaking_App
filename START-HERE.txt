COMMON PROBLEMS & QUICK FIXES:
═══════════════════════════════════════════════════════════════

Problem: "No devices detected"
Fix: Connect phone via USB, enable USB debugging
     OR start Android Emulator first

Problem: "Python not found"
Fix: Install Python from https://www.python.org/downloads/
     Make sure to check "Add Python to PATH" during install!

Problem: "Flutter not found"  
Fix: Install Flutter SDK and add to PATH

Problem: Backend won't start
Fix: Check Windows Firewall, allow port 8000
     Check if another program is using port 8000

Problem: App won't connect
Fix: Check IP address matches in config
     START-APP.bat will detect and fix this automatically!
